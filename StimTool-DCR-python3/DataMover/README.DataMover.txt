README for the Data Mover module

*****************************************************************SUMMARY**************************************************************************

This module just copies data from one location to another and gives an error message if it fails.
This way, data files (including video) can be written locally (fast/reliable) and then copied onto a network location once the session is complete.

*****************************************************************TRIAL STRUCTURE******************************************************************

*****************************************************************INPUT DETAILS********************************************************************

This task uses a somewhat different input format.
Each line in a .schedule file specifies a source and destination separated by a comma.
